h1. CAPS

CAPS (Curation Architecture Prototype Service) is a Django app for lightweight curation of digital objects, powered by curation microservices.  More detail later.

h2. Dependencies

* python (2.6.6)
* django (1.2.3)
* gitdb (0.5.2)
* gitpython (0.3.1)
* xlrd (0.7.1)
* pairtree (0.7.4) - http://github.com/mjgiarlo/pairtree
* arkpy (0.2) - http://github.com/mjgiarlo/arkpy
* bagit (0.8) - http://github.com/edsu/bagit
