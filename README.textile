h1. CAPS

CAPS (Curation Architecture Prototype Service) is a Django app for lightweight curation of digital objects, powered by curation microservices.  More detail later.

h2. Dependencies

* python (2.6.6)
* django (1.2.3)
* gitdb (0.5.2)
* gitpython (0.3.1)
* xlrd (0.7.1)
* "pairtree":http://github.com/mjgiarlo/pairtree (0.7.4)
* "arkpy":http://github.com/mjgiarlo/arkpy (0.2)
* "bagit":http://github.com/edsu/bagit (0.8)
